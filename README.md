 - 画风突转，好久没看过文学氛围这么浓厚的小说。描述人物形象损人不带脏字，基本都是比喻用典；掺杂一点文言文的描述方式
 - 现在已经不太喜欢这种类型的小说了。进行人物虚构，可能会将现实中出现过的多个人物形象进行杂糅，然后为了情节需要，尤其是已有的历史大背景下，故意放大某些缺点，突出特定时代背景下的人物特点
 - 并没有看懂题目“围城”是啥意思，
 - 书中最后附加的一点内容——杨绛写的《钱钟书与《围城》》——更有意思
